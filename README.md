# Oliva Reads - Diario de Lectura Social

Este repositorio contiene la aplicación completa "Oliva Reads".

## Estructura del Proyecto

El proyecto está dividido en dos partes principales:

1.  **Frontend (Directorio Raíz):**
    - Contiene la aplicación de interfaz de usuario construida con React y TypeScript (`index.tsx`, `App.tsx`, etc.).
    - Es un prototipo interactivo y visual que define la experiencia de usuario.
    - Para ejecutarlo, consulta la **GUÍA DE INICIO RÁPIDO** dentro del archivo `App.tsx`.

2.  **Backend (`/backend`):**
    - Contiene la API del servidor construida con Laravel.
    - Se encarga de la lógica de negocio, la interacción con la base de datos, la autenticación de usuarios y la comunicación segura con servicios externos como la API de Gemini.
    - Para ponerlo en marcha, consulta las instrucciones en `backend/README.md`.
